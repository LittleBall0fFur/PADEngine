# PAD Engine
Basic 3D Engine Runs in Vulcan On Windows, Linux and Mac.

How to build
---------

Windows, Linux, Mac:

	mkdir build
	cd build
	cmake -Wno-dev ..
	make
	./PAD

Visual studio:

	mkdir build
	cd build
	cmake ..
	now open the visual studio project
